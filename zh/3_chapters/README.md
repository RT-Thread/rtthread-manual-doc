# RT-Thread移植、驱动开发指南

本章节文档包括两部分：

* RT-Thread移植指南。描述了如何把RT-Thread移植到一颗新的芯片上，其中又会包含两类：针对cortex-m系列的不同芯片间移植；针对全新架构芯片移植。
* 驱动开发指南。描述了，针对一颗新的芯片，如何实现、开发底层的外设驱动，从而适配到RT-Thread系统中。

## RT-Thread移植指南

* ARM Cortex-M芯片类型移植
* 新芯片架构移植
